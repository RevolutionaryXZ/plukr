What is Plukr?

Plukr sets up its own version of the internet much like tor or i2p.

The nodes will be smart phones, servers, raspberry pis, desktops, etc.

You will be rewarded with amounts of Plukr for how long you have your node up.

Anyone would be able to access Plukr. And with the help of the nodes, Plukr will be another internet alternative.

How it works

Instead of using serveral mainframes, and giant server farms owned by Google, Verison, etc, we use the paticipating nodes in a mesh network.

The nodes simiply download our software for their device, leave it running, allow access to the network, and earn Plukr.

Why Plukr?

The Plukr network will be secure and anonymous, like tor or i2p.

This could help if net neutrality is abolished. Or if your site gets taken off the clear web.

Plukr would protect against spying and tracking. And allow you to browse comfortably.
